¿Cómo hemos hecho esta galería?

Esta galería ha sido creada utilizando HTML y CSS para estructurar y diseñar las páginas web. Cada jugador tiene su propia página de detalles que incluye información sobre sus características, perfil y habilidades destacadas. Las imágenes de las cartas de los jugadores se han optimizado para web y se han organizado en una estructura de carpetas para facilitar el acceso.

Hemos utilizado enlaces de navegación para permitir a los usuarios moverse fácilmente entre la página principal de la galería y las páginas de detalles de cada jugador. Además, se ha prestado atención a la accesibilidad y la usabilidad para asegurar que la galería sea fácil de usar para todos los visitantes.

Herramientas utilizadas

  • Editor de código: Visual Studio Code
  • Navegador web: Google Chrome
  • Software de edición de imágenes: Adobe Photoshop
  • Lenguajes de programación: HTML5 y CSS3
